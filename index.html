This is a demo of Debiki Embedded Comments

Read about Debiki Embedded Comments here: http://www.debiki.com/for/embedded-comments

The demo works like so: first there is an example blog post, and below it there is a section with embedded comments powered by Debiki. Feel free to add test comments.
