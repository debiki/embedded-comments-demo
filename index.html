Embedded Comments Demo

A demo site for Debiki Embedded Comments

You can read about Debiki Embedded Comments here: www.debiki.com/embedded-comments

Instructions

Do this:

  1. Have a look at the comments below, and think about whether interesting comments have surfaced to the top, above all boring test comments. — The intention is that you find the most interesting comments first, when you start reading.

    Currently the underlying algorithm simply counts Likes, but I intend to improve on this and for example take into account which comments people have actually read when they click Like.

  2. Test to leave a reply: click Reply.

  3. Suggest an improvement: Click the text in a comment below and select Improve. Then you will be able to edit the comment and submit an improvement suggestion to the author of the comment. (Doesn't work with old web browsers, <= IE8.)
